# graphite
Shortcut bar for wayland

> [!NOTE]
> Even though this bar was made with wayland in mind, it was tested on Hyprland and so if it breaks on your machine, please don't fail to leave an [issue](https://github.com/lordryns/graphite/issues/new) 

### Dependencies 

- clang
- gtk4
- gtk4-layer-shell 
- log.c (included)
- inih (included)

# Installation 

Simply clone this repository and run the `make` command (ensure you have clang and gtk4 installed).
